🔄 REQUEST FLOW
────────────────────────────────────────────────────────────────────────────

1. Browser
   ↓ GET / → /index.html
   ↓ GET /app.js
   ↓ Form submit

2. JavaScript (static/app.js)
   ↓ POST /api/agent/query
   ↓ {"query": "user question"}

3. Go Server (main.go)
   ↓ handleAgentQuery()
   ↓ runReAct() loop
   │ ├─ Call Ollama LLM
   │ ├─ Parse "Action: search_pdf"
   │ ├─ Call searchPDF()
   │ ├─ Parse "Action: web_search"
   │ ├─ Call webSearch()
   │ └─ Repeat until "Final Answer:"
   ↓
4. Response
   ↓ {"answer": "synthesized response"}
   ↓ Display in browser

🔧 EXTENDING THE PROJECT
────────────────────────────────────────────────────────────────────────────

Add CSS:
  1. Create: static/style.css
  2. Link in: static/index.html
  3. Restart server (no rebuild needed)

Add images:
  1. Copy: static/images/logo.png
  2. Reference: <img src="/images/logo.png">
  3. Restart server (no rebuild needed)

Add API endpoint:
  1. Add function in: main.go
  2. Register route: http.HandleFunc("/api/new", handler)
  3. Rebuild: go build -o agent main.go
  4. Run: ./agent

Modify UI without coding:
  1. Edit: static/index.html or static/app.js
  2. Just restart: ./agent
  3. No recompile needed!
